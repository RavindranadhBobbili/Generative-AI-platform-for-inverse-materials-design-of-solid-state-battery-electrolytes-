GenMatDesign-AI

Generative AI for inverse materials design of solid-state battery electrolytes using a Conditional VAE, surrogate modeling, uncertainty estimation, and deployable Flask APIs.

Device: cpu | Models Ready: True | Internal Library Size: 35

1. Train Internal Model

Train the generator and surrogate model using the built-in electrolyte library.

2. Predict One Electrolyte

Enter an electrolyte sequence such as PEO-LiTFSI-SN-Al2O3.

3. Inverse Design from Desired Properties

Specify target properties and generate new candidate electrolyte sequences ranked by closeness to target.

4. Architecture

Built-in Electrolyte Library ↓ Tokenization + Vocabulary ↓ Conditional VAE Generator ↓ Surrogate Property Predictor ↓ Inverse Design from Target Properties ↓ Candidate Ranking + Uncertainty ↓ Dashboard + REST API

5. API

GET /api GET /health POST /train POST /predict POST /generate GET /export

Example /predict payload

{ "sequence": "PEO-LiTFSI-SN-Al2O3" }

Example /generate payload

{ "target": { "ionic_conductivity": 1.75, "stability_window": 5.15, "glass_transition_temp": 29.0, "stability_score": 0.90 }, "top_k": 10 }
This repository is a portfolio-grade inverse materials design prototype for GitHub. The internal library is illustrative and intended for demonstration, not publication-grade discovery.
